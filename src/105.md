# 均码，又名百分之一百の不合身

【开挂】新晋快时尚产品夸奖语，“这一季的 zara/h&m/uniqlo 开挂了！”表达了购买者对少量金钱能买到大量美丽这一事实的欣喜，你不禁理解了在特价酬宾时段买到好吃地瓜并不停跟你表达实在太划算了的你外婆。

【均码】服饰选购避雷针，又名百分之一百の不合身。日式均码像孕装，韩式均码像袍子，欧美均码均的可能是维密天使；同一款均码背心，腰短个矮穿是 oversize，腰长个高是抹胸；均码裤子，壮着胆子买回来，试穿完怀疑自己长了双假腿。

【厦门化】指一种”你大学后面原来卖肉夹馍和麻辣烫的老板改在铺花砖装黄铜色吊灯的撸猫咖啡馆里冲咖啡“式的产业升级。

【进食返祖】一种信奉吃得越接近石器时代老祖宗的饮食习惯越好的中产进食原则，具体表现为，只食用蔬菜水果肉鱼坚果等可直接获得的食物，不碰谷物、乳制品之类的加工食物，迷恋长时间熬制的骨汤原汤，五小时一碗老妈蹄花汤，十八小时精煮慢熬腌笃鲜，好吃到你丧失计算能力，忘记商家开门时间总共不到八小时。

【职业升级下沉】刚刚晋升为部门经理的你回到家乡见到同时操作区块链炒币、母婴产品代购和拍摄 vlog 的家乡亲属后敏锐察觉到而不愿承认的一种结构性变化。

【犬式社交】因为抗拒过年去亲戚和对象家被拉着挨家拜年以及应酬各种不熟的人的社交压力而产生的社交需求：为什么不能像狗一样社交？散步、偶遇，不安排、不刻意，见到的每一面都如此顺其自然。（@来自好奇心研究所智囊团@孙痱子）

【犬式互吹】“天啊你这条裙子好衬你的肤色，也显得你的腰身好细！”“是的，再见。”（@来自好奇心研究所智囊团@孙痱子）
