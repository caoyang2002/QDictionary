# 听国产职场剧讲魔幻现实主义童话

![gif](./assets/qd.gif)

**【品牌故事】** 一种忽悠性质的商业文学。代表性情节：创始人曾经很厉害但也很迷茫最后开了这家店回归初心——“每个人内心都有一个卖花/咖啡/肉夹馍的清新理想，不同的是我做到了（所以你应该给我送钱。”



**【国产职场剧】** 以魔幻现实主义表现手法呈现的当代童话。童话里的职场仙子们做最少的事，住最fancy的公寓，犯蠢不会被辞退反而会触发“你好天真好特别和那些勾心斗角的恶毒同事好不一样”因此男/女一二三四五六七都爱上你的特殊剧情，且最后总有一个匹配的公司内部异性终成眷属。



**【热泪盈眶】** 一种由当代青年自我感动引发的疾病。永远年轻，永远上呼吸道感染。



**【新型肉麻】** 由新兴辣眼事物引发的生理性排斥，病原体多为爱在在微信发木心金句和人生感悟“不嫉妒不纠结不计较不…”，在知乎回答问题以“先说一下自己，我自认长相中上，从小被夸好看，经常被人叫男/女神”为开头的眼前苟且但渴望诗和远方的人群。



**【技术苦手】** 指“怎么调出任务管理器”都要上网查至今没学会海淘在每个无法打开的外链下面留言“求教到底如何翻墙”的成年人类，技术宅的反义词，互联网时代的残障人士。



**【匹配机制】** 一个自我认知偏差矫正机制。打游戏感觉队友都是智障时意识到这是一个匹配机制，不禁陷入深深的思考。



**【政府官网】** 一种界面审美比肩“兰州牛肉拉面”“沙县小吃”“公牛插座五金店”招牌的网站。



**【swag】** 一个由美国说唱歌手发扬光大的扮酷词汇，指一股“叮铃康啷裤链相互撞击，声音总和超过六十分贝”的垮劲儿。



**【nice】** 形容词，用以描述拥有“随时随地让陌生小猫立即跪舔”“地铁上随机收获人类幼子微笑示好”“被踌躇好久的路人看到就会一个箭步上前问路”等超能力的人类。大部分时候看脸，有时候也看摸猫技术。



**【怀孕】** 一场给生活加量加价的大型消费升级，又给了自己一个理由全面浏览更多品类的海淘产品。
