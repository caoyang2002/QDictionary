# 金马奖，内地演员的阳澄湖

![gif](./assets/qd.gif)
**【EP】** 即extended play，即为了保持持续的人气和关注度而产生的，以一个歌手隔三差五发一两首歌为主要形式的音乐产品，是一种“你也没耐心听，我也不愿意投那么多钱做”型转变，一种适应时代的技巧。所以同样听100首歌，搁以前你是听了8张专辑，搁现在就是听了100张专辑，非常丰富的一年啊！宾主尽欢！

**【奥运型歌手/世界杯型歌手】** 四年出一张专辑，其余时间可能在演戏/做生意/搓麻将/谈恋爱/在音乐节上跑调。

**【安可压力】** 包含以下几种情况——1. 表演者已经第三次出来谢幕了而观众已经离场只剩三成；2. 表演者满怀期待观众喊安可而观众回应寥寥；3. 表演者在观众没有喊安可的时候谢幕又回来唱了三首歌；……一场表演者和观众共同承担的尴尬。（本条受到好奇心研究所读者@霁方 的启发）

**【听曲计时】** 以歌曲取代“一炷香”“一盏茶”作为时间计量单位的当代计时法，比如洗一张专辑的澡再敷三首歌的面膜，从容度过规律且悦耳的夜晚。

**【BGM清洁加成】** 一定要在背景音乐加持下干家务才更有效率的一种三次元属性加成现象，具体加成效果视 bgm 品种与个人匹配度而有所不同，有人只有开着麦兜才能施展厨艺，也有人必须边听郭德纲相声集边断舍离。唯一的麻烦是你和你（未来）的伴侣又多了一个潜在分歧——在好不容易解决了谁扫地谁拖地之后，你们有可能还会为了听陈奕迅还是听二手玫瑰再次大打出手。

**【懒人沙发】** 一种吸引懒人往上坐，坐上去变成残疾人，再过一会儿变成死人的“意志剥夺型”家具。

**【浴缸】** 一种“早知当初”的懊恼型家装元素。以为能天天泡泡浴然而最终命运通常以“嫌费水和麻烦堆满了脸盆，只有在洗猫狗时用”而告终。

**【iPhone信任危机】** 指，使用非原装 USB 线连接 iPhone 与 mac 时疯狂弹出无法连接的情形，一种新时代电子信任危机。

**【炫光瞎】** 因为使用了炫光耳机、炫光鼠标、炫光鼠标垫等全套游戏设备，一敲键盘整个世界都闪耀着玛丽苏七彩光芒，而引发的视觉失灵。珍爱生命，远离忽悠型黑科技。

**【颜色角色】** 用以形容固定一类商品的颜色名称。比如说，虽然是相近的颜色，但是你搜卡其色，出来的结果都是裤子；你搜驼色，出来的结果就全是大衣。一种约定俗成。

**【文化衫】** 革命者的最终归宿。逝世的革命者能不能在天堂相遇很难说，但他们一定会在文化衫上重逢，走遍丽江阳朔田子坊和南锣鼓巷。

**【养生型镀金】** 一种力求去除垃圾食品气质的复古广告语，比如一碗米粉是“防辐射，清除体内毒素，净化血液”的，一款红薯干是“富有预防癌症，降低血脂的作用”的……总之从街口的肉夹馍到胡同里羊肉串吃了都有延年益寿的功效，而且多半都是乾隆试吃过的。

**【莲花】** 中年（危机）经典意象。似乎只有换一个莲花头像，转发内核为莫生气的《原作菩萨那朵莲》，佩戴莲花形状的首饰，背面刻心经，才能缓解中年危机带来的情绪暴躁。

**【小牛光环】** 即一种“比起过度自谦，适当吹点小牛反而不让人讨厌”的心理现象，这就是一直声称“我这次考砸了”结果成绩出来却很好的同学比那些爱吹牛结果栽沟里的同学更讨人厌的原因。

**【示爱型勒索】** 以表现“我是如此的爱你”的方式，企图让人在道德/舆论/不知道咋回事的压力下回馈“我也爱你”的一种感情绑架。比如没有感情基础就当众摆蜡烛告白；比如关系压根儿没谱就天天主动道晚安还要求他人道晚安；比如动辄就“我在你楼下，有话好好说”……一种情商低到谷底几乎接近犯罪边缘的讨嫌行为。

**【行事日历】** 一种因为“能合理规划并优雅完成每天的任务”而购买，但实际上每次写都要翻三十多张才能找到当天那页的自带 flag 的效率单品。

**【美剧更新纪周法】** 指一种“以西部世界确定周一、神盾局确认周三，TBBT 确定周六”的资深追剧人士计时方式。

**【金马奖】** 内地演员的阳澄湖。外地野蟹放到阳澄湖一两周就完成镀金变阳澄湖准湖蟹，演技很难说的演员得了金马奖也自动完成了演技入流的认证。一种认证领域的江河日下。（@好奇心研究所辞典特约研究员龚鉴）

**【王室没落】** 一种令人唏嘘的title缩水。如果你仔细回想近几年戴过“亚洲舞王”王冠的男明星，你就能发现这份名单包括但不限于罗志祥、刘承俊、张佑赫、尼古拉斯赵四......“xx 帝”如今也不再尊贵，懂球帝/理财帝/数学帝，这个词的含金量如同它的起源地“贴吧”一样逐渐没落。（@好奇心研究所辞典特约研究员龚鉴）

**【修饰性否定】** 即一个名词前加上一些定语之后，就被迫失去原本含义的一种语言现象。比如微电影和大电影都不是电影；轻小说和微小说都不能算是小说；而号称超人气作家的一般也不是作家，他们甚至都不够人气，还有点气人。

**【眼泪】** 永远不被相信的励志鸡汤陪跑型词汇，如果北上广的职场情场菜市场都以“不相信”眼泪的力度和频次来“不相信”网络诈骗，世界和平指日可待。
