# 苹果，真的有保密协议吗

**【“酷”表情】** 表情包对酷的理解至今停留在“墨镜、叼烟、歪嘴笑”的东北社会人老哥形象。一种装酷文化的历史图腾。



**【配平式影评】** 一种表现为“挺好看的，但是现在竟然有 9 分，我得打个一星中和一下”的胡搅蛮缠，作者通常是菜场公平秤转世托生，奇怪的是中学时期化学方程式从未成功配平。



**【转走】** 微博神社的标准祈福姿势，转走这个锦鲤/大龟/粉海豚/idol笑容/魔法动图，即可好运连连一夜暴富猫狗双全四世同堂桃花朵朵开……可以说是人生巅峰直升梯了。



**【fun culture】** 高级职场招聘英语。主打 fun cultrue 的多半是创业公司 HR（CEO 本人兼任），宣称“我们团队都是 96 年刚毕业小孩，老板整日穿的一副有嘻哈看多了的样子，布置任务都是 yo homie ，这个 case 帮我 handle 一下，你看我们 culture 多么融洽……”其实是公司实在没什么像样的福利。



**【鸿沟】** “你做的这个产品经理是个啥级别的经理”叫代沟，“30 岁还干这种小打小闹的工作不想着赚钱，你还是太年轻！”叫价值观鸿沟。



**【title inflation】** 兴趣特长具象化趋势下的个人 title 通货膨胀，新马泰自由行达人、双十一优衣库抢单圣手、第三届方便小火锅配饭大赛连吃三碗米……屁大点人生经历都能放大成闪光点，在微博个人简介里完美重现了大学时期填写学生会应征简历时的大型抓耳挠腮。



**【安可上瘾症】** 想要被挽留而作势离开的以退为进行为，比如告别演唱会开了上百场，每隔两年还继续加演的歌坛奇迹，近十年内拍的每一部都是息影前的告别之作，公众人物经过关闭评论弃号的好几出戏后又因为“粉丝过于想念”而回归，一种迷恋捧哏相声文化的戏精。



**【浸没式追剧】** 指众网友面对社会热点事件发挥叙事特长充当野生编剧自行为其续写结局的行为，受到众包概念的启发，预测及填补了每一条可能的故事线，一种“就你这种公关水平还不如我来”反套路高手姿态。



**【押韵】** 一种乐感评判标准的转移的现象，以前还是看你音准节奏音高识别，现在乐感的底线是一句话里有双押。



**【参照物】** 初中物理最硬的知识点，教会普通人找大脸人士合照显脸小，启发科技界将产品转接口做大点儿显超薄。



**【WWDC】** 苹果发布会，一种将讲新课演绎成总结知识点的复习性质产品发布，生动诠释了保密协议是一种通过泄密体现存在感的商业文件类型。
