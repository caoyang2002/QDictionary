# 本周份辞典以及我们出书啦！

![gif](./assets/qd.gif)
**【发油】** 不存在安全用量的一种cos道具，多加一滴宛如从不洗头的斯内普上身，少加一滴炸毛程度就像一个头两个大的早年奥普拉。

**【取关帕金森】** “果取关”不到三秒再次订阅，再取关，再订阅......的循环往复行为，表面上看是“我的双手不听使唤”，其实是把对方当对象来爱的分手前求挽留。

**【节气】** 伪装“修身养性有文化”人设必备要素，附庸风雅界的门萨测试指标——是否在微博上扮演过一个定期发布节气美图的农耕时代天气预报员。

**【修图应用】** 养兵千日用兵一时型 app，常用的不超过三个但囤满四屏也不嫌多，万一有朝一日需要做个魔幻效果/漫画化/添加一个别处都没有的河马贴图呢，有备无患。

**【终身会员】** 网络账号无法顺利注销而意外收获的长情终身契约。如果中途毁约，可能需要在犄角旮旯找到和“本公司享有最终解释权”一样神秘的注销守则，严厉拒绝一个月免费会员的惊人福利，在不同设备上确认身份，陪客服唠嗑十天半月...才能脱身，是当代社会比婚姻还稳固的一种契约制度。

**【众所周知的原因】** 通常不是因为众所周知，而是因为众所不能明说。

**【套餐沉没成本】** 打包购买的产品中，并不可心的，派不上啥用场的鸡肋产品的价值，比如十二色眼影盘里从来不用的那十色，adidas 运动袜三件装里那双大红的，团购四人餐里没人要吃的那盘苦瓜，一种从开头就预料到但总是鬼使神差下单的赔本买卖。

**【muji 制造商】** 制造业悟空，全世界最忙、分身最多的家居产品制造者。

**【立享】** 一种文案香氛，主要用于给”首先先关注我们的微信公众号注册会员，留下手机号方便我们日后发送垃圾短信，并把本活动页面分享给一百个好友点亮图标，最后点击抽奖并避开万分之 9999 ‘谢谢惠顾’的概率即可获得 iPhoneX、iPhone8、iPad等壕礼”类优惠介绍营造一种不存在的唾手可得氛围。

**【橱窗】** 比较早期的一种 AR 设备，比起如今的人工智能及算法，橱窗的增强现实功能主要靠人脑达成，比如对着 0 码牛仔裤看见自己不存在的马甲线，对着蒂凡尼总店啃面包嚼出世间美好，人脑对战电脑的一次胜利。

**【邮费补拍定律】** 指“你最想点进去的那张图，不是售罄仅供欣赏，就是邮费补拍”的淘宝店未解之谜。

**【自杀式袭击】** 指，一种义无反顾奔赴预售战场，奋勇拼抢一口气交付十八单定金，只为双十一当天能够高枕无忧（含泪支付数十倍尾款）的英勇行为。（来自研究所智囊团成员@何茫茫）
