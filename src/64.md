# 房价，当代奇幻生活的碳 14

**【999+】** 音乐人的 10w+。


**【Gap】** 指“尬 Rap”，一种当代表演灾难，表演者努力要跟上节奏，舌头失去控制，一只手做着徒劳的“yo yo yo”，歌词一言难尽，配上用力过猛的服装和投入的表情，灾难度和跳街舞被关掉音乐差不多。



**【show trade】** 和对象就“今晚看啥”或“接下来刷啥”产生内部矛盾时互退一步的折中做法，我答应你看血腥暴力，那你也要陪我看爸爸去哪儿，fair enough。这项交易在青年情侣间的成交率较高，而上了年纪的夫妇在商谈破裂后通常都会过上“你在电视上看球，我用 pad 看剧”的分屏生活。



**【CP】** couple pretended，宣传期假戏假作情侣档。



**【微博广告】** 告别韩式半永久，你会迎来可以结交多金帅（gay）气男的相亲平台，不管你相亲是否成功，过一阵就推婚纱摄影，一种无论你点多少次“不感兴趣”也要走完全程的一条龙式骚扰。



**【存在感钓鱼】** 回复“TD 退订”，对方就知道此号码有一个拥有人类情感的正常人在使用；点击“不感兴趣”，平台就知道你是一个连广告都会看的寂寞人士。



**【定制版】** 一种来自商家的一言难尽的消费想象：谁不想过上用河南新闻广播特别定制版充电宝为吴昕定制版朵唯手机充电，阅读罗辑思维定制版《智能时代》为自己充电的绚丽人生呢！



**【正常甜】** 人类意志力的薄弱环节。面对奶茶店店员“您真的要点正常甜吗？正常甜非常甜噢”的询问，多数人会做出“那就七分甜”的妥协。这同样也是人类理性的脆弱环节，显然很少有人思考过如下问题——既然非常甜，那为什么不定名为“非常甜”或“反常甜”？而要（欺诈性地）称呼为“正常甜”？（来自好奇心辞典特约评论员@龚鉴）



**【变身日】** 指在商场打折当天，拧不开瓶盖的柔弱女性突然能成力扛 50 斤购物袋的日子。



**【521】**  给 520 没有来得及准备礼物的朋友一次挽回机会的日子。



**【房价】** 新时代的碳 14，由于房价每年都在创造奇幻新高而使之成为判定年份的参考依据，比方说，“这笑话出来的时候北京房价才三万”，足以表明该段子的生成年份在 2012 年左右。但“房价太贵了”就无法作为判定年代的依据，因为从 88 年开始的报纸里就一直没有停止过房价太贵的吐槽。



**【绝望】** 一种时髦的攀比指标。


**【镜子】** 健身房使用率第一的器械。



**【吸猫片】** 指互联网居民聚众吸食静动态猫图片的成瘾行为。吸过猫片的人通常都是人生嘤嘤嘤家。
