# 洪荒之力再见，“丧”才是互联网 new sexy

![gif](./assets/qd.gif)
**【生态厨房】** 外卖盒饭界的照骗团伙。号称匠人精神十几道工序打造的生态盒饭，生态原始到徒手拌菜、剪刀摆盘，虫鼠与过期食物齐飞。匠人精神被黑得最惨的一次。

**【购物狂欢节】** 一种复杂程度堪比五险一金的电商套路型优惠活动，玩法包括各种各样的积分换优惠券，优惠券满减，满减还分场地，不同场地不同时刻发红包.......其宗旨就是“不可能让你尝到所有甜头”。

**【online window shopping】** 指，下了一堆导购海淘独立电商 app，没事就看看，不买。

**【社交功能偏执症】** 一种多发于迷信社交场景提高打开率、期待自己成为张小龙2.0 的产品经理身上的疾病，表现为执着地给自家 app 增加根本不需要的社交功能，致力于让你边看预报边聊天气、边锻炼身体边讨论健身的时候穿不穿内裤，或者是在钱包里跟朋友们交流感情。

**【热点癌】** 多发于公众号、微博营销号运营负责人的职业病，临床表现为出现新闻热点就集体高潮，再八竿子打不着的内容也要在标题里写上“洪荒之力”蹭一发热点。

**【动图写作】** 发源于 Buzzfeed 的快时尚文风，写作格式为一句话配一幅动图。公众号写手入门必修。

**【扰民式直播】** 指，直播主播抄起一根自拍杆出没于各大音乐节主题乐园发布会现场横扫千军边走边播，严重影响活动体验的扰民行为。

**【丧】** 既“萌”、“贱”之后流行于互联网的 new sexy，爱好者头像来自马男截图，存有全套葛优躺图片和自制“我已经是个废猫了“系列表情，以此传达一种颓丧而优雅的生活态度。

**【董】** 民谣界贵族姓氏。自拍发朋友圈并配文 “X小姐，你不是一个没有故事的女同学” 或 “大梦一场的X二千先生” ，但除非姓董，否则真的显得很硬来。

**【页游广告】** 互联网牛皮癣。
