# 还没看《复联》和《权游》的，统统算作社会闲散人员

![gif](./assets/qd.gif)

【A 爆】饭圈用语，指某位明星有“攻气”，符合追星族不爱好好说话的用语习惯，只是圈外人难免会错意，误以为是哪位明星手里有 “A 炸”，一心只想帮她叫地主。

【Color Run】在 2019 年，Color Run 的过气程度，堪比早年间那些扑通扑通掉下水的竞技闯关。而那些还在一门心思搞 Color Run 的落伍主办方，叫人想要规劝他们：请别再浪费彩色的玉米淀粉。

【复权社会】在这个四月的社会，人们的正事就是要讨论《复仇者联盟》和《权力的游戏》，而那些无法加入讨论的人，会被归为“社会闲散人员”。

【叛逆期】广告代理商们好像正处在一段冗长的叛逆期，要展现年轻人的生活态度，他们的文案总是：是谁说的 xxx 我偏要 xxx 他们说 xxx 我就是 xxx——且不说年轻人的风流不靠这种口舌争斗，一天到晚问人“是谁说的”显得挺耳背倒是真的。

【奥特莱斯】时装品牌的工厂店，如果运气足够好，你能在那里看到各种奇形怪状的失败设计，或是因配色诡异而滞销的丑东西合辑，颇有一种正常零售店受到辐射后变异的即视感。

【拆迁体】“撕掉不时髦的条纹 T”、“敲掉中式卫生间”、“烧掉极简风”，新媒体作者对待过气审美的态度之暴力，雷神再就业抡大锤也不过如此。

【发带】适合舞台装束，但在现实生活中佩戴往往显人用力过猛——尤其是当你戴着发带却告诉别人，你真的不是谦虚，你真的不是街舞男孩也不会跳地板动作，你最多只会一些会了还不如不会的傻乎乎手指舞。

【联名 T 恤】不得不说，一些知名企业搞出来的联名 T 恤乍一看实在太像员工服，我们要怀疑，它的目标客户，是那些潮人男女中比较想进知名外企的那一批。

【铜锈】售后杀手。商家已和消费者达成默契，双方都应该明白金属制品发生氧化是设计的一部分，使用痕迹赋予了物件真正的生命。如果你执意追问为什么黄铜灯买来没多久就变色，对方多半会惋惜你看不懂铜绿的独特，听不出十九世纪的电气之音。

【拿在手里很有质感】如果这个词出现在手机评测中，大约可理解为“重得像一个健身器材”。
