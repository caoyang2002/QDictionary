# 你也是一个煲耳机、养牛仔裤的都市农夫吗？

**【生活美学】** 生活和美学都很难得，但生活美学一百个公众号里有二十个都有信心教会你——在家里摆一百棵绿植，放无印良品熏香，刮掉墙皮露出石砖墙面，喝茶的杯子上也要保留泥土痕迹，并坚信收纳/扫除/吃茶叶蛋也是一种生活美学。



**【绿植】** 一种低成本的复古意象，在一家 Vintage 店里，植物和盆栽必不可少。比起永生不死还不能降解的塑料花，新鲜植物表明店主和顾客已达成默契，双方都明白何为凋谢，何为生命并非永恒。（@好奇心辞典特约评论员朱凯麟）



**【复古押宝】** 一种时尚领域的博彩行为。每年都有复古风回潮，但回档到 70、80 还是 90 年代都很难说，为了紧跟潮流你必须频繁光顾精品店购买二手古着，坚信消亡的风格样式是明年大热，或者后年。（@好奇心辞典特约评论员朱凯麟）



**【新型务农】** 煲（耳）机，养牛（仔裤），做一个都市农夫，了解动铁机动圈机的灵敏度，原牛水洗牛的品种差异，关心猫须和蜂巢，从而实现发自人类本能的清新务农理想。



**【精神辅食】** 婴儿因为没长牙所以只能吃米粉和泥湖状食品，有些成年人因为没长脑所以只能看“知识型视频脱口秀”和“帮你读书节目”。



**【晴空节目】** 一种总是将世界表现为晴空万里、以俗套探案集或鸡汤喜剧为主的过气影视剧类型。



**【雾霾节目】** 现如今为了吸引口味黑化的年轻观众，电视台只有比拼谁的设定更丧气更阴暗以获取关注，比如伦敦咖啡店女老板的忧愁与忧愁、令人嫌弃的过气好莱坞影星的一生、如何证明你不是一场糟心游戏里的倒霉 NPC 等。一个治愈系寒冬。



**【神剧劝退师】** 指，在前两集热烈赞颂神作并向一周目观众殷切提示“这只是铺垫”“撑过前三集就好看了”“你没看出神来是你瞎”的弹幕作者，关了弹幕心平气和觉得还能看，开着弹幕就想弃剧。一粉顶十黑的标准示范。



**【史诗级】** 一个令人怀疑当代史诗其实是以三行情诗来定义的形容词。



**【微创新】** 即，没有创新。打着微创新旗号的公司同时还会提出诸如“C2M：工厂直达消费者”，“重新打磨，致敬经典”，“原厂原单，大牌再现”，“追踪工艺，原料进口，同样品质，更优价格”等广告语，公司理念主要是“以跳过设计师为己任”。微创新的近义词还有：复刻、本土化、祖国版。



**【椰树牌椰汁】** 无论有没有设计师，它都是事实上真正贯彻了“以跳过设计师为己任”理念的品牌。



**【kindle进度条】** 碎片化始作俑者。心流杀手。世界上走得最慢的东西。



**【咖啡馆书架指数】** 一种非正式的阅读趋势大数据，咖啡馆从前流行三毛和《盗墓笔记》，现在基本都有《从 0 到 1》。



**【年会包袱】** 以“为什么要占用我的休息日开年会”、“年会到底穿什么才能像个人样而不过分像个人样”、“我为什么要放弃手上的报告来准备年会上表演的这些傻X节目”、“前年抽到了一条毛巾去年抽到一个U盘如何才能让我有尊严地结束今年年会” 四大心理为主的年末职场心理挑战。



**【自然美景审美衰落】** 形容一种“因为自然美景长期被用作屏保和中年男性头像而令人爱不起来”的当代审美变化。



**【野生帝王型娱乐】** 如果说在用餐的时候欣赏歌舞是帝王般的享受的话，在吃饭的时候观看拉面表演……可能属于野生帝王型享受吧。
