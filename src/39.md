# 闹铃是一种操作简易的傻瓜式时光机

**【表情包式语塞】** 指不用表情包就无法聊天，不把一大堆丑表情加在文章中间就写不出像样文章的语言类疾病。



**【poser】** 本意是装模作样的人，现指什么都要po上网的人，其症状是囫囵阅读完一本书后po上豆瓣同时分享到微博，看个演唱会从晒票到固定机位直播，看一集西部世界发二十张截图到社交网络。



**【创业三心】** 即初心、匠心、业界良心——初心用以形容创始人灵光一现找到的（不存在的）需求，匠心指创始人坚持的“就是因为这个所以我们的产品和服务才那么贵”的产品特色，业界良心指“因为业界都没有良心所以我们很棒”的自我陶醉。



**【借壳视频】** 指内容和标题严重不符的欺诈型视频，看一个罗伯特德尼奥的老片，p1 是电影，p2 是全能家居改造王；搜卡特十佳进球，点进去是 up 主本人野球场蹩脚动作集锦；跟着郑多燕跳操跳得满头大汗，画面一切后半截全是减肥微商广告。一种视频界的挂羊头卖狗肉行为。



**【刷脸卡】** 指智能设备进行面部识别时突然卡住的不智能时刻。



**【迂回搜索】** 早期互联网遗民的搜索习惯——点开主页是hao123导航的浏览器，在页面里的百度搜索栏里敲下“百度”进行搜索，从搜索结果页里进入百度首页，然后搜索“网易邮箱”，至今仍常见于部分事业单位台式机用户。



**【闹铃】** 一种操作简易的傻瓜式时光机，把闹铃摁了再翻个身，闭眼五分钟就能穿越到两个小时后甚至更远的未来。



**【壁挂自行车】** 因为“自行车挂墙上富有现代工业感且能完美结合个人爱好和空间利用率”而尝试的心愿型装饰风格，最终命运通常以朋友圈还没发出去就掉下来，或者因无法解决“骑一次挂回去就会大白墙上留下轮胎印”的清洁问题而成为永久性装饰，自行车本人都忘了自己是个坐骑而告终。



**【洁癖包袱】** 形容一种“别人不干我不开心，别人干了我不放心”的家务分工领域内的心理障碍。（好奇心研究所读者@软绵绵棉花糖—）



**【综艺百家姓】** 以“中国”、“星”、“梦”为首综艺节目常用姓氏，其记忆口诀为”中国歌舞星秀，非常梦想达人”，与之相关的一个数学猜想则是，这些词排列组合一共可以得到多少档综艺节目。



**【大电影】** 以“一定要搞点东西出来挣点钱啊否则对不起这大IP啊”为初衷制作的，不是电影的电影。



**【微电影】** 以“一定要搞点东西出来挣点钱啊否则对不起我这身才华啊”为初衷制作的，不是电影的电影。



**【一万小时购物】** 即将一万小时花在研究品牌、型号、价格乃至工艺、历史上的购物行为，比如为了买洗衣机认真研读波轮、滚筒、洗烘一体、集尘盒、烘干机叠放支架等名词，关注A品牌实则为B品牌代工的工业真相和B品牌最近甚嚣尘上的换CEO风波的商业新闻，恶补化学知识研究衣物面料到洗涤剂的化学成分构成，对比不同购物平台的折扣数字，一场调动全面知识储备的购物行为，是当代人主动获得新知的一大途径。当代人往往经历着践行“一万小时定律”二十年，只修行成功了“一万小时购物”的残酷真相。



**【膨化食品】** 总在紧张工作的间隙怀着来一点压压惊的愿望打开而在袋子没空前再也停不下来的上瘾型食品。一种办公室大麻。



**【完美食欲症】** 即，一种为了获得极致健康而亦步亦趋追随时髦饮食趋势的疾病。患者为了吃得健康而到了偏执的地步：首先，他们去掉了鸡胸肉以外的肉，然后是牛奶里的脂肪，然后是玉米，然后是大豆，然后是麸质……过一段时间当没有东西能吃了，接着他们就改变关于食物的信仰进入一个新的轮回。



**【火车书】** 指火车站售卖的，以鬼故事鸡汤、艳情故事鸡汤和逝世领导人秘辛鸡汤为主的图书类型。



**【机场书】** 指机场售卖的，以成功学鸡汤、佛学禅修鸡汤和“从古到今每一个名人都教你发财型”鸡汤为主的图书类型。



**【地铁书】** 鸡汤本人丢的书。



**【北上广】** 常年处于多事之秋的人类聚居地，命运多舛到让人怀疑这其实是一个名为“北上广”的可怜人，它不相信眼泪，经常要面对人们逃离或杀回的反复无常，偶尔遭遇地铁被丢书的人为肠梗阻。建议发起一个“请不要再北上广搞事情”活动。放过北上广。
