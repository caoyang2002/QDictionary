# 没有卖过日历的杂志社，一定不是好咖啡馆

【漂染】掉色速度之快几乎肉眼可见，以粉色为例，一个月内你能经历“粉色—淡粉—粉黄—枯黄”整条色阶，一次漂染，四次换头，堪称物超所值的美发选择。

【国语综字】为国内综艺节目打字幕的确是精细活——当嘉宾说“杀”，字幕要改成“刷”；当嘉宾说“ rich ”，字幕里要解释为“精神富有；“偶像”要改成“榜样”，就连“坤哥”都要改成实名制的“杨坤”，照这种修改强度来看，字幕人员几乎可以算进翻译工种。

【挂耳咖啡】与日历一起，加入文创产品之“中看不中用”标配套餐。总之在这年头，要是哪家公司没有出过一两款难喝的挂耳咖啡，那它就算不得一家像样的出版商/杂志社/新媒体。

【核心产区】从咖啡豆到白酒乃至橘子皮，总有人鼓吹农作物产地。以前只有阳澄湖大闸蟹才是正宗的，引进核心产区的概念以后，邻居太湖家的大闸蟹身份也贵重了起来，一个藏着扩招技巧的卖货词汇。

【校歌】（几乎）全国的校歌都有一些共同点：节奏都是平稳四四拍；歌词都是“ XX 之滨，XX 山旁，莘莘学子，自省自强”一类暖心寄语；在校学生都不知道怎么唱。

【葱油饼】国产编剧的精神食粮，只因他们对于职场关系、人生境遇的刻画总落入“东宫娘娘摊大饼，西宫娘娘剥大葱”的贫乏想象——每个公司高管都要当众发飙，每个富家子弟都要承包鱼塘，每个女性白领，都要面临家庭事业难两全的终极困境。

【丹宁】“丹宁”和“牛仔布”的区别，应该类似键盘和电子琴的区别——一个听起来更好听些；本质上差不多但如果细说的话可能会有点差；不过也没人在乎差别在哪。

【消失的红点】这正是某些应用软件的鸡贼之处——App 右上角出现莫名其妙的红点，点进去后却发现什么新消息都没有，这突发的红点，更像是手机界面难解的疹子。

【流量教】一种新型宗教。流量神教的中心教义认为，宇宙是由一根发红的大拇指创造的，大拇指说，要有闪光灯，于是就有了闪光灯。教徒毕生以积赞作妖为己任，赞数达到十万加就能在脸皮死后升入天堂。流量神教信仰的天堂里有一艘满载大拇指的火箭，和一套拥有五千模板的滤镜 app 。地狱拥有同样的火箭和滤镜 app，只不过地狱的大拇指都朝下，滤镜则无法美颜。

【无樱花节】整个三四月，全国可能有一千个地方在办樱花节，但一些城市/大学不过寥寥几株樱花竟也要凑这个热闹。开办节日的热情令人怀疑，当樱花凋落时，他们甚至也愿意开一个“无樱花节”。
