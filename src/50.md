# “婚礼现场抢婚”是史上最浪漫无用功，你咋不去民政局拦呢
![gif](./assets/qd.gif)
**【年终奖跳蚤市场】** 指人们在朋友圈、闲鱼等场所出售年会上赢来的电视机、iPad、手机、扫地机器人、生鲜电商购物卡的行为，而购买者手持年终奖，刚好完成年末最后一笔消费升级，非常喜气洋洋的一项交易行为。但要是你每年都抽到手机壳、U盘、充电宝和毛巾，就不怎么喜气洋洋。



**【开箱帖flop】** 原指一类将购买的消费电子产品自包装盒内逐步地解开封装并以相机或是摄影机详细记录的帖子，而现如今除了你想看的第一手数码产品开箱之外，还有大量诸如网易严选优雅格纹羊毛拖鞋开箱和金锣哈尔滨风味红肠开箱的存在，令人感到开箱帖界的泥沙俱下。



**【姻缘运费险】** 一种基于广告打假的旅行需求，当旅行产品冠以“艳遇”大理、“艳遇”波斯的名字的时候，有必要厘清“没遇着艳遇算谁的”这一实际风险。



**【净土型景点】** 以“私藏”“秘境”为噱头的景点，事实上这类经典很可能配套一塌糊涂，住宿只能考虑当地村民家的炕，包车费用占到全程旅费70%，常常号称“再不去就晚了”，但其实什么时候去都太早，再等一轮沧海桑田可能都来得及。



**【情侣旅游胜地】** 景区名为“情人谷”“情侣路”，标配为蹩脚心形建筑/树/大石头，以挂把锁摁个手印为主打项目的情侣尬游胜地。不过说起来如果说真有什么情侣旅游胜地，我希望是一座男友摄影培训基地。



**【纪念品无用指数】** 如果一个机场售卖的都是包装差而非常甜的亚洲点心，辟邪的画和护身符、代可可脂巧克力、八十元档位的丝绸制品和珍珠制品，它的餐厅一定也售卖过咸的汤面和酱汁凝固的加热食物。“纪念品无用指数”和“饮食难以下咽指数”一起构成了衡量机场档次的重要考核元素。



**【新型电子脚镣】** 出差的对象每天给你微信运动点赞，周末出没出门下班有没有直接回家都尽在掌握；你妈发来大众点评截图质问“你不是口腔溃疡吗怎么又在给火锅店五星好评，还报名了八个川菜霸王餐”……“让我一个人静静”或成往事，唯有屏蔽删号保平安。



**【空城】** 即，连外卖都叫不到了的大城市，一种真正的孤独。



**【叶公好你】** 形容每年过年都要经历的一次“父母盼望你回家，而你回家后他们又嫌你懒/丑/胖/玩手机/晚睡/不找对象/找不着对象”的过程，现在父母们的叶公指数越来越高，他们不仅忙着痛斥你“你一回来满地都是饼干面包碎渣！你一回来满家都是你的头发！昨天才打扫完卫生今天就乱成这样子！你能不能不要随手乱堆乱放东西！家里的蟑螂就是你从北京背回来的！”还常常在你最需要爱的时候通知你：“自己去热饭，我在抢红包没空理你。”



**【父爱如山】** 一种富有中国特色的男性育儿思路，即，像山一样静静地坐在那里，观看配偶带娃。



**【母爱】** 行李箱杀手之一。除了托运时不可避免的摔打，你妈在你离家返工时塞进去的番薯白菜、腊肠、风干牛羊肉、活鸡两只和蟹田大米十斤，对你的行李箱总是用不过两年亦有贡献。



**【硬启动】** 当代人长假归来重新开工的自我激活方式，然而不进行心理建设强行回到工作状态，身体往往会通过“定了五个闹铃还是无法成功起床”发出抗议，以示“您的操作有误”。



**【固定笑点flop】** 某些承袭自网络上古时代的，看个开头就能猜到下文的笑点。例如，如果在喜剧电影中，团队中一位成员A要为另一位成员B讲述“团结是力量”的道理，A会给B一根筷子让其折断，A再给B三根筷子，B能折断，A大惊。这种笑点若是出现在《读者》合订本的“每日幽默”版块尚能被接受，但出现在 2017 年贺岁档电影中只能让人怀疑导演是不是近两年没怎么看电视。



**【笑点密集】** 多数时候指的是“全程自嗨”。反过来想，这也未尝不是一种影视剧避雷针型词汇——看到“笑点密集”自动避让，这是尴尬癌的一个我最后的尊严。



**【影视逻辑】** 有违于现实生活中的一切逻辑，影视剧中的人物遵守的是另一套理性准则。如，“去婚礼现场抢婚”可能是史上最浪漫的无用功，因为符合常理的做法应该是在更早些时候尽力阻拦这对恩爱恋人去民政局领证。



**【“xx也是一样”】** 鸡汤文的中心段落。在此之前对一个完全无关的事件进行陈述，接上一句“爱情/人生/......也是一样”后强行扯东扯西。如，“巴西龟并不出产自巴西，人生也是一样，我们总要背井离乡去陌生之地打拼。”



**【godvia】** 巧克力界的潘多拉，因其经济而成为直男情人节送礼物的第一选择。



**【程序员】** 扭转当代爱情故事节奏的人群。随着社会的发展和开放，传统爱情故事里最动人心弦的“追求和暧昧”阶段不得不被挤压，因为观众会发出“都什么时代了，见了八次面还不能表白吗”等呼吁，但当爱情故事的一方主角是程序员的时候，“追求和暧昧”阶段再长都可以理解。从这个意义上说，程序员作为一种感情新物种，开启了当代情爱文学新的春天。



**【新型电子脚镣】** 出差的对象每天给你微信运动点赞，周末出没出门下班有没有直接回家都尽在掌握；你妈发来大众点评截图质问“你不是口腔溃疡吗怎么又在给火锅店五星好评，还报名了八个川菜霸王餐”……“让我一个人静静”或成往事，唯有屏蔽删号保平安。



**【刷牙】** 一种穿越行为，指“刷牙的时候在屋里兜一圈，再在窗前站一会，往往就会产生一种’这都是朕的江山’感”的心理现象。从这个角度讲，牙刷可能是当代成本最低的 ego-booster，十二块五收获每天两分钟的野生帝王体验，可以说很划算了。
